# AIOS Export Specification v1.2 — Release Artifact Specification

Status: Draft

Defines official release artifacts.

## Supported Artifacts
- Markdown
- PDF
- SSOT Bundle
- Knowledge GPT Package
- CHANGELOG
- Release Notes

## Requirements
- All artifacts MUST originate from approved sources.
- Artifact versions MUST be consistent.
- Release bundle MUST include manifest and version metadata.
