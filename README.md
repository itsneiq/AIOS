# AIOS

AIOS (AI Operating System) is the official repository for the AIOS documentation and specifications.

## Repository Structure

- core/ — AIOS Core SSOT
- runtime/ — Runtime specification
- production/ — Production specification
- prompt-standards/ — Prompt standards
- templates/ — Templates
- repository/ — Repository specification
- qc/ — Quality Control manual
- export/ — Export specification
- rfc/ — Requests for Comments
- changelog/ — Release history

## Single Source of Truth

AIOS_Core_v4.3_SSOT.md is the architectural authority. Other specifications derive from the Core and must not redefine it.
