# Security Policy

## Supported Versions

AIOS is currently preparing its first official release.

| Version | Supported |
|---|---|
| v1.0.0 | Planned |

## Reporting Security Issues

Please report security-related concerns through GitHub Issues until a dedicated private reporting channel is configured.

When reporting, include:

- affected document or workflow;
- expected behavior;
- observed behavior;
- potential impact;
- suggested mitigation, if available.

## Response Policy

Security-related reports should be reviewed before release decisions when they affect:

- AIOS authority integrity;
- repository integrity;
- release packaging;
- Knowledge GPT safety;
- documentation governance.
